============
Installation
============


Stable release
--------------

To install Dynamic Green Ammonia, run this command in your terminal:

.. code-block:: console

    $ pip install dynamic_green_ammonia

This is the preferred method to install Dynamic Green Ammonia, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Dynamic Green Ammonia can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/ZackTully/dynamic_green_ammonia

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/ZackTully/dynamic_green_ammonia/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/ZackTully/dynamic_green_ammonia
.. _tarball: https://github.com/ZackTully/dynamic_green_ammonia/tarball/master
